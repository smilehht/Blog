# 二、数据管理

## 数据分析的目的
数据分析是根据目前业务数据总结得出现状、问题、原因、本质、规律等信息，它能帮助：
- 总结数据发现问题
- 向上汇报
- 阶段总结，指导后续运营

## 分析报告基本内容
- 标题
- 结论性文字
- 关键指标数据及指标对应的口径解释
- 关键指标变化趋势图标(图表)

## 数据分析指标体系
“无法衡量，就无法改进”，通过搭建数据分析指标体系，对核心指标进行统计分析，及时发现异常变化，对不同渠道进行分析处理，让运营策略更有策略性和针对性。

## 图表使用
- 柱形图：对比分析
- 折线图：趋势分析
- 饼图：比例分布
- 散点图：分布分析
- 漏斗图：转化率分析
